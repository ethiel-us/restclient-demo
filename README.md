# restclient-demo

Laboratory project for **AIISS 2026**

## Description

Demo project showcasing the use of a REST client in Java with Spring Boot.

## Technologies

- Java
- Spring Boot
- Maven

## Prerequisites

- Java 17 or higher
- Maven 3.x

## Installation & Running

1. Clone the repository:
```bash
   git clone git@github-university:ethiel-us/restclient-demo.git
```

2. Navigate to the project folder:
```bash
   cd restclient-demo
```

3. Run the application:
```bash
   ./mvnw spring-boot:run
```
